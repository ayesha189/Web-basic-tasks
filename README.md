# 💻 Web Development Daily Tasks

A growing collection of daily web development practice projects — built to sharpen frontend skills through real-world mini-applications.

## 👩‍💻 About This Repository

This repository contains my daily web development tasks and projects. Each project is built from scratch to practice and improve skills in **HTML**, **CSS**, and **JavaScript**. New projects are added regularly as I progress through my learning journey.

---

## 🚀 Projects

### 1. Simple Calculator
A clean, functional calculator built with HTML, CSS, and JavaScript.

**Features:**
- Basic arithmetic operations — addition, subtraction, multiplication, division
- Clear and delete buttons
- Responsive button layout
- Real-time display updates

---

### 2. ATM Machine
A simulated ATM interface that mimics real banking operations.

**Features:**
- PIN authentication system
- Check account balance
- Deposit money
- Withdraw money with balance validation
- Transaction error handling (insufficient funds, invalid input)

---

### 3. Forex Trading Dashboard
An interactive foreign exchange trading dashboard UI.

**Features:**
- Live currency pair display
- Buy and sell interface
- Exchange rate visualization
- Clean, professional trading UI layout
